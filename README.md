# 🔐 Himangi — Cybersecurity & AI Portfolio

A modern, cybersecurity-themed personal portfolio website with Matrix rain animations, terminal-style UI elements, and a sleek dark aesthetic. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## ✨ Features

- **Matrix Rain Canvas** — Animated background inspired by _The Matrix_, rendered on an HTML5 canvas
- **Typewriter Effect** — Dynamic typing animation on the hero section
- **Terminal Card** — Interactive Kali Linux–styled terminal showcasing bio info
- **Scroll Reveal Animations** — Sections animate into view on scroll
- **Responsive Design** — Fully responsive across desktop, tablet, and mobile
- **Contact Form** — Built-in form with client-side validation
- **Glassmorphism UI** — Frosted-glass card aesthetics with subtle borders and glows
- **Dark Theme** — Cybersecurity-inspired dark palette with neon green accents

## 🖥️ Getting Started

### Prerequisites
Any modern web browser (Chrome, Firefox, Safari, Edge).

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/himanggii/Portfolio.git
   cd Portfolio
   ```

2. **Serve with any static server**
   ```bash
   # Using Python
   python3 -m http.server 8080

   # Or using Node.js
   npx serve .
   ```

3. **Open in your browser**
   ```
   http://localhost:8080
   ```

Or simply open `index.html` directly in your browser — no build step required.
